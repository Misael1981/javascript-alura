# Tipos de Dados em JavaScript: Um Resumo Completo

O JavaScript possui dois tipos principais de dados: primitivos e estruturados . Cada um deles tem características e usos específicos, e compreendê-los é fundamental para escrever código JavaScript eficiente e robusto. 

## 1. Tipos de Dados Primitivos:

Os tipos de dados primitivos representam valores imutáveis, ou seja, que não podem ser alterados após a criação. São os tipos mais básicos da linguagem e incluem:

- **Numbers**: Números inteiros (ex: 10, -25) e decimais (ex: 3. 14, 1. 5e2).
- **String**: Sequências de caracteres entre aspas simples ('olá') ou duplas ("mundo").
- **Boolean**: Valores lógicos que podem ser trueou false.
- **Undefinid**: Indica a ausência de um valor.
- **Null**: Representa a ausência intencional de um valor.

## 2. Tipos de Dados Estruturados:

Os tipos de dados estruturados permitem organizar e armazenar conjuntos de dados mais complexos. Os principais tipos estruturados em JavaScript são:

- **Object**: Coleções de pares chave-valor, acessíveis por meio de pontos ou colchetes (ex: ). `{ nome: 'João', idade: 30 }`
- **Arrays**: Listas de valores ordenadas, acessíveis por meio de índices (ex: ). `[1, 2, 3, 'quatro']`
- **Function**: Blocos de código reutilizáveis ​​que podem receber entradas e retornar valores (ex: ). `function soma(a, b) { return a + b; }`
